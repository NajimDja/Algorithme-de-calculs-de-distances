# Algorithmes de calculs de distances

Expérimentation d'algorithmes de calculs de distances tel que Dijkstra et A*, qui permettent de calculer le chemin le plus court entre deux points et découverte de la librairie NetworkX qui permet de créer des graphs.

## Intsallation des dépendances

Il est conseiller de créer un environnement avant d'installer les librairies nécessaires à l'utilisation du repository qui sont dans le requirements.txt

1. Clonez ce repository :

```bash
git clone https://github.com/NajimDja/Algorithme-de-calculs-de-distances.git
```

2. Intallez les dépendances :

```bash
pip install -r requirements.txt
```
